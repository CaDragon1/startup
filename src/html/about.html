    About

    This website is a practical implementation of a personal progress tracker linked to a Google calendar, created by Caleb Drake. The end goal is to, someday, expand upon this concept to harness addictive app design into an app that encourages users to set and accomplish goals, create a daily journal, and improve themselves.
